# CSS組版のテンプレ

## 必要なもの

* Node.js

## 使い方

1. パッケージのインストール

```
npm install
```

または

```
yarn
```

2. ビルド

```
yarn build
```

また、以下のコマンドでscssファイルとhtmlファイルを監視して自動でビルドする。
(index.htmlからpdfが生成される)

```
yarn watch
```


## 使用したもの

* [vivliostyle](https://vivliostyle.org/ja/)
